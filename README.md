Minhas Tarefas!
Este projeto é uma aplicação web de gerenciamento de tarefas (To-Do List) desenvolvida para a disciplina de Programação Web, do curso de Análise e Desenvolvimento de Sistemas (ADS) na Uninter.
A aplicação permite o controle total de tarefas cotidianas, utilizando persistência de dados local e uma interface responsiva.

Funcionalidades:
CRUD Completo: Criar, Ler, Atualizar e Deletar tarefas.
Persistência Local: Os dados são salvos no localStorage do navegador.
Filtros Dinâmicos: Filtragem em tempo real por Status (Pendente/Concluída) e Prioridade.
Níveis de Prioridade: Diferenciação visual entre prioridades Alta, Média e Baixa.
Interface Responsiva: Adaptada para Desktop e dispositivos Mobile.
Visual: Animações de entrada e saída, e marcação de tarefas concluídas com tachado.

Tecnologias Utilizadas:
HTML5: Estruturação semântica.
CSS3: Estilização com Flexbox e animações.
JavaScript: Manipulação do DOM, High-Order Functions e Web Storage.
Font Awesome: Ícones vetoriais para ações de interface.
